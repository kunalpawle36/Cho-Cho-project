# Cho-Cho-project

This is a complete analysis of a chocolate company's sales and operational data.

## Overview
The project analyzes comprehensive data from a chocolate manufacturer, including sales performance across different regions, product categories, and promotional campaigns. The dataset is structured into dimensional and fact tables for efficient analysis.

## Data Structure
The dataset includes:
- **Dimensional Tables**: DimChannel, DimDate, DimGeography, DimProduct, DimPromotion
- **Fact Table**: FactSales containing transaction-level sales data

## Purpose
This structured data supports business intelligence and decision-making for optimizing sales strategies, understanding market trends, and improving inventory management across geographical regions.
